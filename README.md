# JOS (Johney Operation System)

Executive-level operating system UI for daily decision-making, communication, and business execution.

## Features
- Business / Private Life OS
- Top Action Dashboard
- Importance / Urgency Matrix
- Omnichannel Communication Hub
- Real-time Action Engine (connected to backend API)

## Tech Stack
- Next.js
- React
- Tailwind CSS
- JOS API backend (.NET)

## Run
```bash
npm install
npm run dev
```
